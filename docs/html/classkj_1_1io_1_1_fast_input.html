Detailed Description

Fast buffered input reader for competitive programming.

Provides zero-allocation, branch-lean methods to read integers, strings, and raw bytes from stdin using a large static buffer. Works in C++17+.

Usage:

kj::io::FastInput in;
int n; in.read(n);
long long x; in.read(x);
std::string s; in.read(s);
