Detailed Description

Manual stopwatch-style timer for measuring elapsed time.

The Timer class allows explicit control over when timing starts and stops. It can also be queried for current elapsed time without stopping it.

Example:

kj::Timer timer;
timer.start();
compute();
double ms = timer.stop();
