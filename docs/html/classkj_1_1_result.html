Detailed Description

template<typename T, typename E = std::string>
class kj::Result< T, E >

A lightweight alternative to std::expected for value-or-error return semantics.

This template class represents a result that either contains a valid value (T) or an error (E), implemented using std::variant. It is designed to simplify error handling without relying on exceptions.

Template Parameters
TThe type of the expected value.
EThe type of the error (default: std::string).
Note
The value type T and error type E must be distinct.
