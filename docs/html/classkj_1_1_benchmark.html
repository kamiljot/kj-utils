Detailed Description

Utility class for benchmarking callable operations.

This class provides a way to perform warmup iterations (to stabilize CPU cache, JIT, etc.) and then collect repeated time measurements using std::chrono::high_resolution_clock.

Results include individual samples and a computed average duration in milliseconds.
