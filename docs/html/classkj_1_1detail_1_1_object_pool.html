Detailed Description

template<class T>
class kj::detail::ObjectPool< T >

Fixed-size object pool with free-list and geometric block growth.

This pool provides fast allocation/deallocation for trivially relocatable node-like objects. It does not call constructors at block allocation time; objects are constructed with placement-new in create and destroyed in destroy.

Template Parameters
TObject type managed by this pool.
Note
All outstanding objects must be destroyed before calling release_all. In DEBUG builds an assertion will fire if the pool still owns live objects.
