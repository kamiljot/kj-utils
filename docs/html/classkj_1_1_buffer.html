Detailed Description

template<typename T>
class kj::Buffer< T >

A simple RAII wrapper around aligned heap-allocated memory.

This class manages a typed, contiguous block of memory allocated with a specific alignment requirement. It cannot be copied but can be moved. Memory is automatically released on destruction.

Template Parameters
TThe element type stored in the buffer.
