Detailed Description

template<typename T>
class kj::Buffer< T >

A simple RAII wrapper around aligned heap-allocated memory.

Manages a typed, contiguous block allocated with a specific alignment. Non-copyable, movable. Memory is released automatically on destruction.

Constructs a buffer with the given size and alignment.

Allocates memory for size elements of type T, aligned to alignment. If size == 0, no allocation is performed and data() == nullptr.

Parameters
sizeNumber of elements to allocate.
alignmentDesired byte alignment. Defaults to max(alignof(T), alignof(void*)).
Exceptions
std::bad_allocif allocation fails.
