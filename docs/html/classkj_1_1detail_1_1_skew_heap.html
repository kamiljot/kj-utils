Detailed Description

template<class T, class Comp = std::less<T>, bool UsePool = false>
class kj::detail::SkewHeap< T, Comp, UsePool >

Skew heap (mergeable heap) with optional internal node pool.

By default this is a min-heap via std::less<T>. Use std::greater<T> for a max-heap. If UsePool is true, node allocations are served from an internal ObjectPool, which can dramatically reduce allocation overhead for heavy workloads.

Template Parameters
TKey type.
CompComparator (StrictWeakOrder), defaults to std::less<T>.
UsePoolIf true, use ObjectPool for node storage (default: false).
